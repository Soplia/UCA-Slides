Interface Lifecycle.SingleUse

  • Enclosing interface:
    Lifecycle

    public static interface Lifecycle.SingleUse
    Marker interface used to indicate that the instance should only be used once. Calling Lifecycle.stop() on an instance that supports this interface will automatically call Lifecycle.destroy() after Lifecycle.stop() completes.
